# About Whiskey Proof Index

## The Most Comprehensive Whiskey Batch Database

Whiskey Proof Index is a free, community-driven resource dedicated to cataloging bourbon and whiskey batch information. Our database includes proof values, release years, age statements, batch numbers, and distillery details for hundreds of whiskey releases.

### What We Track

Our comprehensive database includes:

- **Proof Values**: Exact alcohol proof for each batch and release
- **Batch Numbers**: Official batch identifiers (seasonal releases, numbered batches, etc.)
- **Release Years**: When each whiskey was released to market
- **Age Statements**: Official age information when available
- **Distillery Information**: Producer and brand details
- **TTB Label Data**: Links to official government label approvals

### Coverage

We maintain detailed records for whiskey from major distilleries including:

- Buffalo Trace Antique Collection (BTAC)
- Heaven Hill (Elijah Craig, Henry McKenna, etc.)
- Old Forester
- Wild Turkey (Russell's Reserve, Kentucky Spirit, etc.)
- Four Roses
- Jim Beam Small Batch Collection
- And many more bourbon, rye, and American whiskey producers

### Why This Resource Exists

Whether you're a collector hunting for a specific batch, a whiskey enthusiast researching proof variations, or simply curious about whiskey releases over the years, this index provides accurate, searchable information to help you find exactly what you need.

### Data Quality

- All entries include verified proof values and release information
- TTB label approvals linked when available for verification
- Regular updates as new batches are released
- Community contributions welcome through GitHub
